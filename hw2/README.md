##Задание №1##

'Какой сейчас год?'.
Если посетитель вводит '2015', то выводить 'Вы правы!', 
если что-то другое -- выводить 'С луны свалился? 2015!'.

##Задание №5##
Перепишите if..else с использованием нескольких операторов '?':

```javascript
var name = 'admin', text;
if (name == 'admin') {
	text = 'Hi';
} else if (name == 'manager') {
	text = 'Hello';
} else if (name == '') {
	text = 'No login';
} else {
	text = '';
}
```
