'Какой сейчас год?'.

## <a name='task5'>Задание №5<a>
Перепишите if..else с использованием нескольких операторов '?':

```javascript
var name = 'admin', text;
if (name == 'admin') {
	text = 'Hi';
} else if (name == 'manager') {
	text = 'Hello';
} else if (name == '') {
	text = 'No login';
} else {
	text = '';
}
```
